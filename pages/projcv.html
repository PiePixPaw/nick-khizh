Purpose of Creation:

I decided to create this website as a personal initiative following the completion of a course on HTML and CSS. My main goal was to deepen my knowledge and understanding of basic web development principles. Additionally, I recognized it would be beneficial for my resume, showcasing my web development skills.

Creation Process:

• HTML

I began by developing the overall structure of the site, outlining the basic structure in a notepad, and then proceeded to create the HTML code.

• CSS

Next, I focused on visual design, creating a preliminary layout of the site in Photoshop. After selecting a suitable design, I started adding CSS styles. During the process, I decided to change the color scheme to something lighter and more vibrant. Additionally, as I added styles, I populated the site with content, including textual content.

• JavaScript

To enhance the visual appeal of the site, I utilized JavaScript, borrowing code snippets from various sources online. This allowed me to implement features such as a scroll-to-top button and image enlargement in the "Certificates" section.

Challenges Encountered:

While creating the site, I faced various challenges, including typos in the code, difficulties with block positioning, and challenges in implementing certain elements without JavaScript.

Website Testing:

Upon completing the primary work, I commenced testing the desktop version of the site. I identified several issues and compiled a list of bug reports. Subsequently, I addressed these issues by conducting bug fixes based on my observations.

Future Enhancements:

In the future, I plan to expand the functionality of the site, improve its multi-platform optimization (particularly the mobile version, which is currently unsatisfactory), and enhance my JavaScript skills by attempting to write several scripts independently.
